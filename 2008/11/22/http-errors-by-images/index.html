Http Errors by Images

Nov 22nd, 2008

There are many cases that HTTP errors occurs with tedious text like “Access Denied” or “Page Not Found”. If we add an image for illustration, it’s more interesting. Even people will like to get HTTP errors to see such these interesting images :)).

400 – Bad Request

401 – Unauthorized

402 – Payment Required

403 – Forbidden

404 – Not Found

405 – Method Not Allowed

406 – Not Acceptable

408 – Request Timeout

409 – Conflict

410 – Gone

411 – Length Required

413 – Request Entity Too Large

414 – Requested URL Too Long

415 – Unsuported Media Type

416 – Requested Range Not Satisfied

417 – Expectation Failed

To me the most interesting one is: 411 Length Required :))
